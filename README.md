# KutuphaneOtomasyonu

Bu proje, C# kullanarak geliştirilen bir kütüphane otomasyon sistemidir. Kullanıcı dostu arayüzü sayesinde kütüphane işlemlerini kolay ve etkili bir şekilde yönetmenizi sağlar. Sistem, çeşitli bölümlerden oluşmaktadır:

Özellikler

Üye Yönetimi
Üye Ekleme: Yeni üyeleri sisteme kaydedebilirsiniz.
Üye Listeleme: Sistemdeki mevcut üyeleri görüntüleyebilirsiniz.
Üye Düzenleme: Mevcut üye bilgilerini güncelleyebilirsiniz.

Kitap Yönetimi
Kitap Türleri Ekleme: Farklı kitap türlerini sisteme ekleyebilirsiniz.
Kitap Türleri Listeleme: Mevcut kitap türlerini görüntüleyebilirsiniz.
Kitap Türleri Düzenleme: Var olan kitap türlerini güncelleyebilirsiniz.

Emanet Kitaplar
Emanet Verme: Üyelere kitap emanet edebilirsiniz.
Emanet Listeleme: Emanet verilen kitapları görüntüleyebilirsiniz.

İade Kitaplar
İade Alma: Emanet kitapların iade işlemlerini gerçekleştirebilirsiniz.
İade İşlemleri: İade edilen kitapların kaydını tutabilirsiniz.
